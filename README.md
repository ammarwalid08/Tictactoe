# Tic Tac Toe

A classic Tic Tac Toe game with a clean, modern UI built as a desktop app using Python and Tkinter.

## Features
- 🧑‍🤝‍🧑 2 Player mode
- 🤖 vs CPU mode
- 🎨 Styled UI with custom fonts and colors

## Built With
- Python 3 + Tkinter

## How to Run
python tictactoe.py
